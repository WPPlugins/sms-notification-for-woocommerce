Sends SMS notification for WooCommerce Order statuses and facility to send bulk SMS to all users using gateways like twilio,clickatell & Nexmo.


== Description ==

SMS Notification for WooCommerce plugin will allow you to send SMS notification automatically for orders placed in WooCommerce. Buyer and Admin both can get SMS notification after WooCommerce  order is placed. All SMS Notifications text can be customized from wordpress admin panel. This is a only WooCommerce extension which includes most popular SMS gateways like Twilio, Clikatell and Nexmo in single plugin to send Order SMS Notification.

This plugin is completely free with all features included so, there is no Pro version of this plugin.

### Features

* Three most popular SMS gateway included - Twilio, Clikatell and Nexmo
* Sends SMS notification everytime while update of order status.
* Fully Customizable SMS text for each status message
* Sending Order Details ( order no, order status, order shop name and order amount and many more shortcodes supported) in SMS text.
* Facility to view complete SMS history along with its status.
* Facility to send Bulk SMS to all customers of your site
* Facility to send Manual SMS. - Send SMS anybody even who is not part of your site
* Different SMS send corresponding to different WooCommerce Order Status.
* Works with WooCommerce 2.0+




== Installation ==
How to install the SMS Notification for WooCommerce plugin

Please follow the steps for plugin installation: 

= VIA WordPress =
1. After download plugin, do login to your WordPress admin interface
2. After login click on *Plugins* from the left hand menu
3. Click on *Add new*
4. Click on *Upload*
5. Browse to the directory you downloaded the plugin to and click *Install Now*
6. After Wordpress has finished unpacking the file click on *Activate Plugin*
7. After the plugin has been activated you will notice a new section  "SMS NOTIFICATION" in WooCommerce settings link. 
8. All Done!

= VIA FTP =
1. After your download unzip woocommerce-order-sms-notification from your download .zip
2. Open your FTP client
3. Browse to /your_wp_instalation/wp-content/plugins/ on your hosting server
4. Uplaod SMS Notification for WooCommerce into this directory
5. Go to your wordpress admin panel
6. Go to plugins and activate WooCommerce Order SMS Notification plugin
7. After the plugin has been activated you will notice a new section  "SMS NOTIFICATION" in WooCommerce settings link. 
8. All Done!
